# Teensy-Motivator
Teensy 4.1 Based Droid Control

# Input/Outputs
Inputs:
    - Single Input: SBUS-16 or SBUS-24
    - Web Interface via Wi-Fi (Trigger actions, etc)
    - Serial Console for debugging and test bench work

Outputs:
    - LEDs: ws2812b
    - Serial (1): DFPlayer music trigger
    - Serial (2): Pololu Maestro or Similar (MARCduino, etc)
    - Serial (3): Pololu Maestro or Similar (MARCduino, etc)
    - Can Bus (1): Robotic Actuators, etc
    - Can Bus (2): Robotic Actuators, etc
    - RS485: Robotic Actuators
    - Web Interface via Wi-Fi (status messages, etc)
